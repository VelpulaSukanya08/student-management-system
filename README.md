# 🎓 Student Management System Pro

A Python-based Student Management System developed using Python and JSON.

## ✨ Features

- Auto Roll Number Generation
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Student Statistics
- Sort Students by Marks
- Export Student Data to CSV
- JSON File Storage

## 🛠️ Technologies Used

- Python
- JSON
- CSV
- File Handling

## ▶️ How to Run

1. Open the project in VS Code.
2. Run:

```bash
python main.py
```
